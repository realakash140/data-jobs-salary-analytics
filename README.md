# Global Data Jobs Salary Analytics

## Project Overview

This project analyzes global data job salaries using an end-to-end data analytics workflow.

The dataset was sourced from Kaggle and processed using MySQL running inside a Docker container. A star schema data model was designed to support analytical reporting.

The final dashboard was built using Power BI to analyze salary trends across job roles, experience levels, and countries.

---

## Tools & Technologies

- MySQL
- Docker
- Power BI
- DAX
- SQL
- Data Modeling

---

## Dashboard Insights

The dashboard provides insights into:

- Top paying data job roles
- Average salary by country
- Salary growth by experience level
- Employee distribution across experience levels
